Textarea HTML

Un textarea es un input con mayor capacidad de texto

Con el min-width al 100% se hace qeu el textarea ocupe todo el ancho del contenedor

Resize

Con resize se puede establecer en que direcciones se puede redimensionar el textarea

Con el valor both se puede redimensionar en ambas direcciones

Con el valor none se bloquea el textarea para que no se pueda redimensioar

Con el calor horizontal solo se puede redimensionar de manera horizontal

Con el valor vertical solo se puede redimensionar de manera vertical

Propiedades

La propiedad readonly hace que el contenido del textarea solo se pueda leer mas no modificar

La propieadad disabled desactiva el textarea para que no se pueda modificar ni se envie su contenido cuando se haga envio del formulario

La propieadad maxlenght establece la cantidad maxima de caracteres que puede ingresar el usuario. Para el siguiente textarea es de 5 caracteres, pruebe

La propiedad rows establece la cantidad de lineas que el usario puede ingresar antes de que aparezca el scroll bar
