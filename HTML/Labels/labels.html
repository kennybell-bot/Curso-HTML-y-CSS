Labels HTML

Sirven para etiquetar inputs, son elementos en línea, es decir que el margin y el padding no son modificables. Se puede hacer que al hacer click en el label se resalte el input al que está asociado. Esta asociación se puede dar de dos formas:

Asociación por id

Para esto se le asigna un id para el input, posteriormente se le pone este id como valor al atributo for del label

Asociación poniendo el input dentro del label

Para esto, se pone el input dentro de las etiqutas label, para esto no hace falta que el input tenga un id ni que el label tenga un for
