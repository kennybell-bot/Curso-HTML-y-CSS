Accesibilidad web

Consiste en hacer la página web navegable para personas con discapacidad haciendo que el DOM generado por el navegador sea fácilmenten interpretable para la tecnología de acesibilidad (como por ejemplo la extensión Chrome Vox).Hacer la página accesible hace que tenga un mejor pocisionamiento. Atributo alt imágenes

Poniendo la descripcion de la imágen como valor del atribut alt, chrome vox va a leer dicha descripción cuando se le haga click a la imagen

Atributo aria label enlaces

Con este atributo, se puede poner una desciipción del enlace para que sea leida por el asistente de voz cuando se le ponga en raton encima.
