Select, datalist y option

Select

Sirve para elgir una opción entre varias, se utiliza la etiqueta de nombre select y para cada una de las opciones se utiliza la etiqueta option

Dentro la etiqueta option, el valor del atributo value va ser el valor que se le enviará al backend y la etiqueta name será el que identifique al elemento en el backend

A los elementos de la etiquta option no se les puede modificar el padding y el margin, si al caso el background

Datalist

El datalist es una mezcla entre el input y el select, se usa con la etiqueta datalist. Se puede asociar a un input asingnandole un id al datalist y poniendo este id como valor en el atributo list del input. el atributo name es el que identificar el elemento ante el backend
