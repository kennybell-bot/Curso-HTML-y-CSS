Audio y video HTML

Video

Para poner videos se utilizan las etiquetas video, como valor del atributo src se le pone la ruta del video.

Se puede poner un mensaje en medio de las etiquetas para cuando el navegador no soporte el video.

  • Para que el video se reproduzca automáticamente se ponen los atributos autoplay y muted.
  • Con el atributo controls se puede poner la barra de tiempo del video, el botón para pausar el video y el control de volumen
  • Con el atributo loop se puede hacer que el video se reproduzca en bucle

A los videso se les puede poner fondos con el uso de CSS

Etiqueta track

Es la etiqueta que nos permite poner subtitulos, se pone entre las etiquetas video

  • En el atributo src se pone la ruta del archivo vtt de los subtitulos
  • Para que los subtitulos apereczcan por defecto se pone el atributo default (este no lleva valores)
  • Con el atributo srclang se puede decir el idioma en el que estan los sutitulos poniendo su abreviacion como valor
  • Con el atributo label se puede modificar el texto del idioma de los subtitulos (el que aparece debajo de la palabra subtitulos)

Los estilos de los subtitulos se cambian con javascript o editando el archivo vtt

Audio

Para pooner un audio se usan las etiquetas audio, con la ruta del audio como valor del atributo src

Tambien se le ponen los atributos controls, para que aparezcan los controles
