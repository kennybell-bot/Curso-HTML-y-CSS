Selectores por clase

Para este se le asigna un nombre a la etiqueta con el atributo class y se utiliza el . seguido del nombre para seleccionarlos en la hoja de estilos

Selectores por id

Se le asigna un id con el atributo id, este id es unico para el elemento, no es recomendabel hacer esto asi porque puede generar problemas despues con el JavaScript, se usa # seguido del id que se le asigno
