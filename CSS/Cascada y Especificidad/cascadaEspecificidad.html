Especificidad

La esepecificidad es la importancia que se le da a cada forma de aplicar estilos, todo esto para resolver conflictos en aquellos escenarios en los que se le esté asignando diferentes estilos de diferentes formas a un mismo elemento de HTML. Existen tres tipos de especificidad:

Orden

Este es el escenario en el que se tienen estilos (con selección por etiqueta) de la misma importancia sobre un mismo elemento. Se presenta una especificidad de (0,0,1) y en este caso el estilo que se aplica es el último que se haya especificado

Selección por clase

Este presenta un nivel de especificidad (0,1,0) es decir tiene mas importancia que el anterior

div ejemplo

Selección por id

Este tiene un nivel de especificidad de (1,0,0)

div ejemplo

Combinando las selecciones, se obtienen especificdades mayores, por ejemplo si selecciona por clase y por id entonces se obtiene una especificidad de (1,1,0)

Jerarquía de estilos

La jerarqquía de las formas de aplicar estilos permiten definir el comportamiento de la cascada de los estilos. Se presenta la siguiente lista ordenada de tal forma que los últimos items son los mas importantes, es decir estos sobrescriben los estilos que se hayan aplicado de las formas superiores:

  • Estilos por defectoi del navegador
  • Personalización del navegador del usuario
  • Estilos de la hoja de estilos teniendo en cuenta las especificidades
  • Etiqueta style
  • Atributo style
  • ! important: Este es un valor que se agrega en la hoja de estilos a la propiedad que se desee
