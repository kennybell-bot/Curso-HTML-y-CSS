Colores CSS

RGBA

Son los colores compuestos por R(red), G(green), B(blue) y A(alfa) que es la transparencia

rgb(3, 206, 164)

Hexadecimal

Es la misma idea del RGB, pero para este caso cada color recibe un numero en sistema hexadecimal

#FB4D3D

HSL

En este sistema se tienen tres parametros

  1. Ubicacion del color en el circulo cromatico (los grados del angulo)
  2. Saturacion
  3. Iluminacion
hsl(47.4, 81.17%, 56.27%)
