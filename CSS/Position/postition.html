Position CSS

Relative

Con esta propiedad se puede establecer la ubicación de un elemento tomando estableciendo como referencia su contenedor o su pocisión inicial si este no tiene contenedor

top

Ubica el elemento en la parte superior de su contenedor o encima de su pocisión original

bottom

Ubica el elemento debajo de su pocisión original

left

Ubica el elemento a la izquierda de su pocisión original

right

ubica el elemento a la derecha de su pocisión original

z index

Sirve para cambiar el orden de los elemento en el eje z, enb otras palabras para modificar como se superponen los elementos

Absolute

Con esta propiedad se establce la pocisión de un elemento tomando como referecia el elemento raiz y poniendo valores a las propiedades top, bottom, right y left. Cuando se le aplica este valor el elemento ya no ocupa un lugar en la página.
