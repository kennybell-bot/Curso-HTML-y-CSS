Metodología BEM

(Bloque - Elemento - Modificador)

Bloque Elemento Modificador

Esta medtodología permite la manipulación de estilos CSS de tal forma que el código sea:

  • Resutilizable definiendo un estilo base y aplicándolo a diferentes elementos con el uso de modificadores.
  • Legible, especificando la funcionalidad de cada elemento en el nombre de sus clases, y sin problemas de especificidad

Bloque

Es todo aquel elemento que no necesita de otros elementos para existir o tener sentido y por lo tanto puede ser reutilizable en culquier lugar de la página y cuantas veces se necesite. Para aplicar estilos a este tipo de elementos, se les asigna clases cuyos nombres expresen de manera precisa sun naturaleza

<bloque class="nombreBloque"> </bloque>

Elemento

Son los que componen al bloque. Para la aplicacion de estilos, se sugiere asignarles una clase cuyo nombre se componga del nombre del bloque al que pertenecen seguido de dos rayas al piso y el elemento que es.

<bloque class="nombreBloque">
<elemento class="bloque__nombreElemento"> </elemento>
<elemento class="bloque__nombreElemento"> </elemento>
</bloque>

Modificador

Son aquellas clases que serviran para establecer estilos particulares a elementos y bloques según sea la necesidad. Para su nombramiento se usa el nombre del elemento que se modifica y la característica que se le esta aplicando.

<elemento class="bloque__nombreElemento nombreElemento--caracteristica"> </elemento>
