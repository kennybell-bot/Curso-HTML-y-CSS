Selectores Avanzados

Selectores por atributo

Este tipo de selección permite aplicar estilos a un elemento dependiendo del valor del atributo especidficado. Se puede elegir por todo el valor, por como termina el valor y por como empieza el valor.

Todo el valor del atributo

Para esto se hace uso de corchetes dentro de los cuales se va a poner el atributo y el valor especificado.

Atributos cuyos valores terminen de cierta manera

Para esto se usan también los crochetes, pero esta vez se pone el signo $ antes del igual

Aatributos que empiecen de una forma determinada

Para esto se usa el símbolo ^ antes del igual dentro de los corchetes.

Selectores por herencia

Este tipo de selección permite aplicar estilos a un elemento dependiendo de dónde esté contenido o que elementos lo anteceden

Selectores descendientes

La primera forma es seleccionar un elemento contenido dentro de otro independiente de a que nivel se encuentra

Ejemplo de un párafo dentro de un span dentro de un div

Para aplicar estilos a elementos que únicamente estan en el primer nivel de contenencia se utiliza el simbolo de mayor que >

Ejemplo de un párrafo que es hijo de un div

Selectores adyacentes

Este tipo de selección sirve para elegir el siguiente elemento que esta en el mismo nivle de jerarquía, esto se logra poniendo un signo + entre los selectores en la hoja de estilos, si se hace con varios elementos de manera consecutiva los estilos se aplicaran al último

Selectores hermanos generales

Esta sirve para seleccionar todos los elementos que esten al mismo nivel de jerarquía, estos se consigue poniendo el signo ~ entre los elementos
