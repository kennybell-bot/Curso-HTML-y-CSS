What is grid?

Grid layout is a way to order elements in screen. It´s a line pattern wich generates a grup of columns and rows, aditionally it can be posible to specify the space between cells usig the gap property.

Creating grid

Defining grid

To definea grid layout, we need to identify the container element and on its CSS rule define the property display with the value grid. Then we can create the grid columns using the property grid-template-columns and as value we put the each column value.

.container{
display: grid;
grid-template-columns: 100px 100px;
}

fr units

This units has the function to distribute the avaliable space among the columns or rows. Each fr unit represents one unit of the avaliable space.

.container{
display: grid;
grid-template-columns: 1fr 2fr;
}

Repeating tracks listing

Indicate the number of columns or rows as value becomes hard when they too many, for this purpose it is used the CSS function repeat(). The function recieves two parameters, first the number of times we are going to repeat the second parameter, in this scenary the size of the column or row.

.container{
display: grid;
grid-template-columns: repeat(2 , 1fr);
gap: 5px;
}

Implicit and explicit grid

The diference between implicit and explicit grid come in that the first one is defined by the user using the prpoerties and tools. On the other hand, implicit grid refers to the definition of new rows when the content overflows the grid. Those new rows or columns are autosized, however using the grid-auto-columns or grid-auto-rows, we can configure the new tracks size.

.container{
display: grid;
grid-template-columns: repeat(3 , 1fr);
gap: 5px;
}

The minmax() function

This CSS function avoids to define a minimun and maximun size for a track. It makes dinamic for changes on the content.

.container{
display: grid;
grid-template-columns: repeat(2, 1fr);
gap: 5px;
grid-auto-rows: minmax(100px, auto);
}

As many columns as will fit

To achieve generate the number of columns that the container size allows, we use the CSS functions repeat() and minmax(). As first repeat function parameter put the value auto-fit, it provides the characteristic to generate as many columns as it is posible. For the next parameter, we use the minmax() function, its parameters are going to be the minimum column size and then 1fr to distribute space.

.container{
display: grid;
grid-template-columns: repeat(2, 1fr);
gap: 5px;
grid-auto-rows: minmax(100px, auto);
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

Line base placement

The grid lines are numbered on the text direction. Based on those lines, we can define the any object beginning and end line. For this purpose, we use the properties grid-column-start, grid-column-end, grid-row-start and grid-row-end. On the other hand, we have a shortcut for each track, grid-column and grid-row. To indicate the element beginning and end, we put them as value with a / between them.

.gridItem{
grid-column: 1/3;
grid-row: 1;
}

Grid template areas

The grid template areas its another way to display the elements. It consists on set up a template using words and then asign the word to element that is going to be on that grid area. However there are some rules to keep in mind to use this type of position.

  1. Every must be filled, it means every cell in the template must have a word.
  2. To span an element just repeat the word the number of cells that it ocupes.
  3. To indicate that a cell is going to be empty it is going to ber marked with a period (.)
  4. This just avoids rectangular grid areas.
  5. The areas can´t be repeated at different parts.
.container {
display: grid;
grid-template-areas:
"header header"
"sidebar content"
"footer footer"
;
grid-template-columns: 1fr 3fr;
gap: 20px;
}

.header {
grid-area: header;
}

.article {
grid-area: content;
}

.aside {
grid-area: sidebar;
}

.footer {
grid-area: footer;
}

Nesting grids and sugrid

We can nest elements with grid layout. Aditionally we can use the subgrid value to align the subgrid tracks whith its parent grid tracks. This value is asigned to the grid-template-columns or grid-template-rows property.

.container {
display: grid;
grid-template-columns: repeat(4, 1fr);
grid-template-rows: repeat(1, 1fr);
gap: 10px;
}

.subgrid {
grid-column: 1 / 4;
grid-row: 2 / 4;
display: grid;
gap: inherit;
grid-template-columns: subgrid;
grid-template-rows: 2fr 1fr;
}
