Object fit y position

Object fit

Esta propiedad de css permite establecer como el contenido de un objeto (imagen o video) se va redimenzionar dentro de su contenedor. La forma disipocision del contenido esta dada por el valor que tenga la propiedad.

fill

Este valor hace que el contenido se redimensione para llenar al contenedor sin respetar su relación de aspecto.

elemento{
object-fit: fil;
}

contain

Con este valor el contenido se redimensionará respetando su relación de aspecto hatas que llene en alto o ancho el contenedor.

elemento{
object-fit: contain;
}

cover

Valor que hace que el contenido crezca respetando su relación de aspecto hasta que llene el contenedor

elemento{
object-fit: cover;
}

none

Es el valor por defecto y hace que el contenido se muestre tal y como es.

elemento{
object-fit: none;
}

scale-down

Este valor hace que el contenido se redimensione como contain o none de tal forma que tenge el menor tamaño posible.

elemento{
object-fit: scale-down;
}

Objecto position

Es la propieadad que nos permite determinar la alineación del elemento dentro de la caja.

Valor

elemento{
object-position: center;
}
elemento{
object-position: top;
}
elemento{
object-position: bottom;
}
elemento{
object-position: left;
}
elemento{
object-position: right;
}

Pixeles y porcentajes

elemento{
object-position: 10px 20px;
}
elemento{
object-position: 10% 20%;
}

Combinación

elemento{
object-position: top left;
}
elemento{
object-position: bottom 10%;
}
