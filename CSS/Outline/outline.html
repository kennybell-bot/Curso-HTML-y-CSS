Outline

Outline es una propiedad de CSS la cual consiste en un tipo de borde que no genera ninguna alteración al tamaño de la caja, esta propiedad tiene parametros para modificar ancho (outline-width), estilo (outline-style) y color (outline-color).

Outline width

Esta propiedad puede tomar los siguientes valores.

elemento{
outline-width: 5px;
}

valor

elemento{
outline-width: thin;
}

thin

elemento{
outline-width: medium;
}

medium

elemento{
outline-width: thick;
}

thick

Outline Style

Esta propiedad permite establecer el estilo con los siguientes posibles valores.

elemento{
outline-style: solid;
}

solid

elemento{
outline-style: dotted;
}

dotted

elemento{
outline-style: dashed;
}

dashed

Outline Color

Esta es la propiedad que permite cambier el color del outline con los siguientes valores

elemento{
outline-color: #ffff;
}

valor
