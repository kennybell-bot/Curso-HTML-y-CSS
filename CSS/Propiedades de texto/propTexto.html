Propiedades de txto CSS

Color

Sirve para cambiar el color del texto

font-family

Sirve para cambiar la fuente del texto, las que fuentes que tienen espacios entre ellas se ponen entre comillas simples, se puede poner varias fuentes separadas por coma para tenerlas de respaldo en el caso de que la primera opcion no cargue

font-size

Sirve para cambiar el tamaño de la fuente

font-weight

Sirve para cambiar el grosor de las letras, con bold se pone en negrita

font-style

Sirve para poner el texto en italicas o oblicuo(inclinado)

text-align

Sirve para alinear el texto a la izquierda(left), derecha(right), justificarlo(justify), centrarlo(center)

text-decoration

Sirve para subrayar(underline), tachar(line-throught),poner unas linea encima del texto(overline)

line-height

Sirve para el interlineado, se puede usar conm porcentajes o numeros

letter-spacing

Sirve para manipular el espacio interletrado

text-transform

Sirve para poner todo el texto en mayusculas o minusculas y poner la primera letra de cada palabra en mayuscula
