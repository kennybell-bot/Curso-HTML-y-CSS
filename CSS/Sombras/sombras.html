Sombras CSS

Hay tres tipos de sombras:

  1. Sombras de caja
  2. Sombras de texto
  3. Sombras de elementos

Sombras de Caja (box-shadow)

Son sombras que se aplican a una caja, se utiliza la propiedad box-shadow con los 5 parametros que son:

  1. Direccion horizontal, como en el plano cartesiano, derecha es mas, izquierda es menos
  2. Direccion vertical, como en el plano cartesiano, arriba es mas, abajo es menos
  3. Difuminado de la sombra
  4. Cuanto se expande el color
  5. Color

Sombras de texto (text-shadow)

Son sombras que se aplican a textos, se utiliza la propiedad text-shadow con los parametros

  1. Direccion horizontal como en el plano cartesiano
  2. Direccion vertical como en el plano cartesiano
  3. Difuminado
  4. Color

Sombras a elementos (drop-shadow)

Sonm para aplicar sobre elementos como imagenes con los siguientes parametros

  1. Direccion horizontal como en el plano cartesiano en 0px siempre
  2. Direccion vertical como en el plano cartesiano en 0px siempre
  3. Difumiando
  4. Color
