Box Sizing

Consiste en darle un tamaño fijo a una caja con el fin de que independientemente si agrego mas padding, border o content, esta ajustara estos parametros para que el tamaño de la caja siempre sea el mismo

Content box

Consiste en establecer un tamaño fijo para la caja del contenido, sin tener en cuenta el padding y el border, es decir que siempre que agregue alguna de las dos ultimas el tamaño de la caja crecera, pero si agrega mas contenido esto no aumentara

Border box

Parta este caso, si se tendra en cuenta el el padding y el margin ademas del contenido, es decir cada ves que el alguno de estos o el contenido cambie, el navegador acomodara todo para que el tamaño de la caja no crezca

El tamaño por default es de 100px por 100px
