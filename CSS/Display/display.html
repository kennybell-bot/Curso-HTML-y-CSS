Display es una propiedad de css que permite establecer la dispocisión de un grupo de elementos.

Block

Esta propiedad hace que los elementos se dispongan en manera de bloque, ocupando todo el ancho disponible y apilandose unos encima de otros. Esta permite que se modifiquen valores de padding y margin, asi como tambien alto y ancho

Inline

Permite disponer los elementos linealmente, es decir uno al lado del otro. Aquí no hay posibilidad de modificar ancho ni alto, además el margin y padding solo se podrán modificar de manera horizontal

Inline block

Dispone los elementos como en inline pero conservando las propiedades de la dispocisión block, es decir que si se podrán modificar valores de alto y ancho, así como también el margin y el padding serán alterables en todas direcciones.

None

Propiedad con la que se hace que un elemento no ocupe ningún espacio en la página y no sea visible para ningún tipo de selección.
