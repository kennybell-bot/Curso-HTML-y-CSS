Emmet

Emmet es un plugin que sirve para mejorar y agilizar el trabajo de maquetación HTML y de estilso con CSS. Sun sintaxis se conforma de elementos que sirven como atajos para generar etiquetas y propiededes dependiendo de cada caso

Operadores Anidados

Hijo

Este operador (>), hace que se puedan poner elementos dentro de otros (anidar elementos), para construir así un árbol de jerarquías. Para hacer eso se sigue la siguiente sintaxis

elemlementoPadre>elementoHijo

Es importante no utilizar espacios

Lo anterior genera la siguiente estructura html:

<;elementoPadre>
<elementoHijo> </elementoHijo>
</elementoPadre>

Hermanos

El signo (+) permite crear dos elementos uno después del otro.

elemlementoPadre+elementoHijo
<elemento> </elemento>
<elementoHermano> </elementoHermano>

Escalar

El símbolo (^) permite crear un elemento en el nivle superior del árbol desde un nivel inferior

elemlementoPadre>elementoHijo^elementoHermano
<elementoPadre>
<elementoHijo> </elementoHijo>
</elementoPadre>
<elementoHermano> </elementoHermano>

Multiplicación

Con el símbolo (*) podemos especificar la cantidad de veces que necesitamos que se cree un elemento.

elemento*3
<elemento> </elemento>
<elemento> </elemento>
<elemento> </elemento>

Agrupación

Con el uso de paréntesis se pueden agrupar elementos.

(elemento1 + elemento2)*2
<elemento1> </elemento1>
<elemento2> </elemento2>
<elemento2> </elemento2>
<elemento2> </elemento2>

Operadores de atributos

ID

Con el simbolo # se puede establecer el identificador del elemento.

elemento#identidicador
<elemento id="identificador"> </elemento>

Clase

Copn el punto (.) se puede establecer la clase del elemento. Para poner varias clases se deben poner separadas por puntos

elemento.clase1.clase2
<elemento class="clase1 clase2"> </elemento>

Atributos Modificados

Con el uso de corchetes se puede asignar un valor a un atributo esepcifico del elemento, indicandolo en su interior.

elemento[atributo="valor"]
<elemento atributo="valor"> </elemento>

Numeración de items

Con el símbolo $ se pueden enumerrar los items que se generen. Ademas de ello se puede establecer la cantidad de cifras de la numeración conm la cantidad de sigono $ que se ponga. Finalmente se puede alterar el incremento de la numeracion con el modificador @ poniendo un valor base o haciendo que esta se decremente agregandole el signo menos al lado.

Numeración con 2 crifras: elemento.item$$*3
Numeración en decrecimiento: elemento.itmem$$@-*3
<elemento class="item01"> </elemento>
<elemento class="item02"> </elemento>
<elemento class="item03"> </elemento>

<elemento class="item03"> </elemento>
<elemento class="item02"> </elemento>
<elemento class="item01"> </elemento>

Texto

Con el uso de las llaves se puede indicar que texto va dentro del elemento.

elemento{texto}
<elemento"> Texto </elemento>
