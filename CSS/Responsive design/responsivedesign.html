Responsive design objective is to make the web page layout adaptable to different devices, for exmaple the following image has the property max-width with the value 100% to don't lose its size, even if its container grows mora than its width.

Media queries

CSS characteristics that prevent some styles from being applied depending on whether a condition is true or not. For responsive design, they are used to apply specific styles when the webpage is displayed on different devices. The point where a media query is inserted is called a breakpoint. For example, the following styles won't be applied until the conditions in parentheses are true. The word 'screen' indicates that the CSS rule is applied to the screen presentation (styles can be applied to print design with the word 'print').

@media screen and (min-width: 80rem) {
.container {
margin: 1em 2em;
}
}
