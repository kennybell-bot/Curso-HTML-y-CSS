Flujo de texto CSS

White-space

Esta propiedad de CSS sirve para manejar los espacios en blanco de un texto (espacios y saltos de linea) dependiendo del valor que se le haya asignado.

nowrap

Este valor hace que en el texto no hayan saltos de línea.

elemento{
white-space: nowrap;
}

white-space: pre;

}
