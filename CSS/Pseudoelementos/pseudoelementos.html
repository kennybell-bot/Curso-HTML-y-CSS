Sirven para aplicar estilos a ciertos aspectos de un elemento, para aplicarlos se coloca el nombre del elemento seguido de dos veces los dos puntos y finalmente el pseudoelemento, en la hoja de estilos.

elemento :: paseudoelemento{
estilos
}
