Herencia CSS

Algunas propieadades de CSS son heredadas, es decir un elemento que esta contenido dentro de otro toma como valor de una determinada propiedad el que tenga su contenedor. Sin embarogo, esto solo pasa con elementos que no tengan un valor definido por defecto.

Valor inherit

Este valor sirve para establecer que el elementotome el valor del contenedor, este irá buscando este valor de forma ascendente y si no encuentra valor alguno toma el que le de el navegador por defecto

Este es un ejemplo de un párrafo que está dentro de un div con fuente Lora en itálica

Valor initial

Este valor sirve para expresar que el elemento conserve su valor por defecto, es decir evitar que este herede de sus contendores

Este es un ejemplo de un párrafo que está dentro de un div con fuente Lora en itálica, pero sin embargo no la aplica, porque tiene el valor initial.

Valor unset

Este valor restea la propiedad a su valor heredado

Este es un ejemplo de un párrafo que está dentro de un div con fuente Lora en itálica, esta si se aplica porque la hereda del div.

Valor revert

Este valor hace lo mismo que el anterior, la diferencia radica en que si el usuario tenga una personalización en el navegador, el valor que tomará será el de dicha personalización

Este es un ejemplo de un párrafo que está dentro de un div con fuente Lora en itálica, la aplica porque respeta la fuente que tiene el div cuando tiene el valor revert.
