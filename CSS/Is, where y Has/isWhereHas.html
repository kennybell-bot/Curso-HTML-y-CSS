Is, where y has CSS

Is Where Has

Son pseudoclase que sirven para aplicar elementos dependiendo de si están contenidos dentro de otros o también de quienes las sucedan

Is

Sirve para seleccionar elementos que estén contenidos dentro del elemento que pongamos antes de los dos puntos, este cambia la especificidad a la hora de aplicar los estilos

Por ejemplo este párrafo cambia de estilo por que es de clase cambio

Where

Sirve para seleccionar elementos que estén contenidos dentro del elemento que pongamos antes de los dos puntos, este NO cambia la especificidad a la hora de aplicar los estilos

Por ejemplo el div cambia de color porque este párrafo es de clase cambioColor2.

Has

Este sirve para aplicar estilos al elemento en el caso de que el escenario entre paréntesis se cumpla, por ejemplo que contenga elementos con cierta clase o id, o que sea sucedido por cierto otro elemento de cierta clae en específico.

Por ejemplo el div cambia de color porque contiene este párrafo de tipo ejemploHas

Por ejemplo el div cambia de color porque está sucedido de este párrafo de tipo ejemploHas
