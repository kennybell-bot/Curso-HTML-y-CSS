Trancisiones CSS

Propiedades

Para hacer trancisiones se utiliza la propieadad transition de CSS dentro de los estilos del elemento que se desea animar. Para indicar que aspectos del elemento se va a cambiar durante la trancisión (por ejemplo color, margin, etc) se utiliza transition-property, para manejar la duración y delay de la trancisión se utilizan trnasition-duration y transition-delay respectivamente. Adicionalmente habrá que indicar cuando se va a dar la trancisión, esto haciendo uso de pseudoclases.

elemento{
transition-property: color;
transition-duration: 2s;
transition-delay: 1s;
}
elemento:hover{
color: red;
}

Movimiento

Para las trancisiones que hacen movimiento, se expresa en la propiedad transition-property la dirección en la que se de va a dar el desplazamiento del elemento, además se configura la duración y delay de la trancisión como ya se vió. Para establecer la forma en que se va dar la trancisión, es decir las velocidades con las que se va a mover el elemento en cada momento del desplazamientose usta la propiedad property-timing-function, esta propieadad tiene los siguienmtes puede tener diferentes valores:

elemento{
transition-property: left;
transition-duration: 2s;
transition-delay: 1s;
transition-timing-function: ease;
}
elemento:hover{
left: 10px;
}
