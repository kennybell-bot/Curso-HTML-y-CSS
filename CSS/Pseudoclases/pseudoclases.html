Pseudoclases CSS

mouse listas elementos inputs

Las pseudoclases son palabras clave que sirven para seleccionar un elemento en un estado específicop y aplicarle estilos. La sintaxis de las psuedoclases es el elemento seguido de dos puntos y la pseudoclase que se vay a aplicar.

Pseudoclases mouse

Son pseudoclases que sirven para que el elemento seleccionado se muestre con diferentes estilos dependiendo de las acciones del mouse

hover

Esta pseudoclase sirve para aplicar un estilo cuando el puntero del mause se encuentre sobre el elemento seleccionado

div ejemplo que cambia de color cuando se le pone el mouse encima

active

Pseudoclase que sirve para dar un estilo cuando se de click sobre el elemento en cuestión

Ejemplo div que cambia de color cuando se le da click

Pseudoclases de elementos

Estas pseudoclases son aplicables a cualquier tipo de eleemtos

not(clase)

Pseudoclase que permite aplicar estilos a todos aquellos elementos que no sean de la clase de los especificados ente parentesis.

root

Permite aplicarle estilos al elemento raíz es decir al html

Psudoclases de enlaces

Son pseudoclases que permiten aplicar estilos a los enlaces dependiendo su estado

link

Pseudoclase que asigna estilos a enlaces que no han sido visitados

visited

Es la pseudoclase que permite aplicarle estilos a los enlaces que ya fueron visitados

Pseudoclases input

Son pseudoclases que permiten asignarle estilos a un input dependiendo su estado

checked

Sirve para aplicarle estilos a un input de tipo checkbox cuando este este chequeado

invalid

Aolica estilos a un input cuando se le esten ingresando datos invalidos

valid

Aplica estilos a un input cuando es formato de los datos que se le esté ingresando es válido
